# David Dew Mallick - Portfolio

A highly animated, responsive, and performance-optimized portfolio website built with Next.js, Tailwind CSS, and Framer Motion.

- **Dynamic Animations**: Smooth page transitions, scroll reveals, and hover effects using Framer Motion.
- **Interactive UI**: Custom cursor trail, tilt effects on project cards, and animated skill badges.
- **Dark/Light Mode**: Seamless theme switching with a custom Sage Green color palette.
- **Responsive Design**: Fully optimized for all devices, from mobile phones to large desktop screens.
- **SEO Optimized**: Implements JSON-LD structured data, semantic HTML, and Open Graph tags for maximum visibility.
- **Performance**: Static site generation (SSG) for lightning-fast load times.
